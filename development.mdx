<Info>

  - Python 3.12
  - Poetry (Python package manager)
</Info>

## Installation for local development

1. Clone the repository:
```bash
git clone https://github.com/felipemeres/ai-analyst.git
cd ai-analyst
```

2. Setup the development environment:
```bash
./setup-dev.sh
```
